## 已知Bug
若某分类下无文章的话，其分类页面的分类名称与slug均无输出，这是Typecho一直没有解决的问题，暂时无解。

## 注意事项
- 侧边栏的缩略图来自文章里的附件中第一张图片（该图片可以不插入文章），建议使用CDN
- 关于页面和分类页面需要手动选择模板新建页面
